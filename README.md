# Personal theme for conky

## Dependencies
- conky
- python2

## Installation
- `git clone https://github.com/Debelzak/deb-conky-theme.git`
- `cd deb-conky-theme`
- `sudo cp -r ./ConkyWeather.ttf /usr/share/fonts/truetype/`
- `sudo ln -s ./conky-startup.sh /usr/profile.d/conky.sh`
- `sudo reboot`
